=== Widget para añadir la IP Pública de nuestros visitantes ===Contributors: MavkSoft.esDonate link: https://www.mavksoft.es/Tags: widget, ip, publica, texto, visitantes, mavksoft, tuipgratis, muchomasqueunblog, pluginRequires at least: 1.0Tested up to: 4.4.5Stable tag: trunkLicense: GPLv2 or laterLicense URI: http://www.gnu.org/licenses/gpl-2.0.htmlEste Plugin nos muestra en cualquier lugar que se pueda añadir un Widget de texto la IP de nuestros visitantes.== Description ==Con este plugin podemos dar el servicio a nuestros visitantes de ver su ip pública, y conseguir de esta forma aumentar nuestras visitas.== Installation ==1. Subir el archivo a wp-content/plugin2. Activar plugin3. Ir a Apariencia > widgets4. Arrastra el widget al sidebar que quieras.5. Comprueba que la IP Pública se ve correctamente!
6. Hay que desactivar para el Widget el caché, ya que sino la IP no se actualiza correctamente.    - Widget: Podemos añadir el Widget en cualquier barra que nos permita el uso de Widget Texto.   - Replace tags: Podemos mostrar la IP Pública añadiendo en cualquier parte [vdd_ip_publica] .   - Template: También podemos añadirlo en cualquier parte de la plantilla con el  codigo php `<?php echo vdd_ip_publica(0); ?>` . == Screenshots ==1.
